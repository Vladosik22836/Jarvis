# J.A.R.V.I.S – AI голосовий асистент (WPF)

Футуристичний десктопний AI-асистент, натхненний системою J.A.R.V.I.S з Iron Man.  
Створений на **C# + WPF**, з анімованим інтерфейсом, boot-секвенцією та основою для голосового керування.

---

## 🚀 Огляд

J.A.R.V.I.S — це десктопний AI-асистент, який імітує роботу інтелектуальної системи в реальному часі:

- Анімація завантаження системи (boot sequence)
- Анімоване AI-ядро (реактор)
- Інтерфейс голосового керування (мікрофон)
- Панель швидких команд
- Журнал системних подій
- Плавні анімації та переходи

Мета проєкту — створити **кінематографічний AI-інтерфейс для Windows**.

---

## 🎬 Функціонал

### ⚡ Система завантаження (Splash Screen)
- Кінематографічна анімація запуску системи
- Ефект друку назви `J.A.R.V.I.S`
- Анімація реактора (пульсація + світіння)
- Вивід системних повідомлень у стилі термінала
- Фонова музика (стиль Iron Man)
- Автоматичний перехід до головного вікна

---

### 🧠 Головний AI-інтерфейс
- Центральне AI-ядро (обертове кільце)
- Індикація режиму "слухаю"
- Кнопка мікрофона з пульсацією
- Футуристичний темний UI

---

### 🧩 Панель швидких дій
Швидкий запуск системних функцій:
- Відкрити блокнот
- Відкрити калькулятор
- Запустити браузер
- Показати поточний час
- Очистити журнал подій

---

### 📜 Журнал подій
- Логування всіх дій системи
- Відображення команд користувача
- Консольний стиль інтерфейсу

---

### 🎤 Голосове керування (UI готовий)
- Кнопка мікрофона для майбутнього голосового вводу
- Підготовка до інтеграції:
  - розпізнавання мови (Speech-to-Text)
  - система команд
  - AI-відповіді

---

## 🛠️ Технології

- C# (.NET / WPF)
- XAML (UI інтерфейс)
- MediaPlayer (аудіо)
- WPF анімації (DoubleAnimation, Transform, Opacity)
- Асинхронні задачі (async/await)

---

## 🎞️ Процес запуску системи

1. Запуск застосунку  
2. Поява splash screen  
3. Анімація реактора  
4. Вивід системних повідомлень  
5. Поява назви J.A.R.V.I.S  
6. Фінальний спалах  
7. Перехід у головний інтерфейс
